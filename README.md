# 📊 Game Data A/B Testing Project
모바일 퍼즐 게임 **Cookie Cats**의 A/B 테스트 데이터를 활용하여 게임 내 gate(허들) 위치 변경이 유저의 retention에 미치는 영향을 분석한 프로젝트입니다.

---

## ✨ Key Findings

> gate_40은 gate_30 대비 더 낮은 retention을 보였으나, 이 차이가 통계적으로 유의하다고 판단할 충분한 근거가 확인되지 않았습니다. 현재 분석 결과만으로 gate 위치 변경이 retention에 유의한 수준의 영향을 준다고 결론을 내리기 어렵다고 판단되며, 로그 데이터 기반의 추가 행동 분석이 함께 이뤄진다면 retention 개선으로 이어질 수 있는 요인을 구체적으로 확인할 수 있을 것으로 예상됩니다.

## 📌 느낀 점

이번 프로젝트를 진행하며 데이터의 구조와 제공 범위가 분석 방향에 큰 영향을 준다는 점을 느꼈습니다.
   
초기에는 EDA를 통해 새로운 인사이트를 찾고자 다양한 파생 변수를 생성하며 분석을 진행했지만, 실제로 활용할 수 있는 주요 변수는 retention과 sum_gamerounds 중심이었기에 분석 결과 또한 기존 변수를 설명하는 수준에서 크게 벗어나기 어려웠습니다.
   
이 과정에서 목적 없는 탐색에 지나치게 몰입하면 오히려 분석 방향을 잃을 수 있다는 점을 몸소 느꼈고 현재 데이터로 검증할 수 있는 범위와 한계를 판단하는 과정 또한 데이터 분석에서 중요한 역량이라는 점을 배울 수 있었습니다.
   
또한 단순 집계 데이터만으로는 유저 행동의 원인을 충분히 설명하기 어렵다는 점을 느꼈고, 실제 서비스 분석에서는 로그 데이터와 같은 세부 행동 데이터의 중요성을 다시 한번 느낄 수 있었습니다.
