# 📊 Data-Driven A/B Testing Project
> 게임 데이터 기반 A/B 테스트 분석을 통해 유저 행동 변화를 검증하고, 데이터 기반 의사결정 방향을 제안하는 프로젝트입니다.

---

## 🎯 목표

- 통계를 단순 이론이 아닌 **실험 기반 분석 도구**로 이해
- A/B 테스트를 통해 **유저 행동 변화 검증**
- 데이터 기반으로 **실제 제품 개선 방향 도출**

---

## 💡 핵심 인사이트

- 유저 경험 초반의 friction이 retention에 중요한 영향을 미침
- 전체 평균보다 **유저 세그먼트별 분석이 더 의미 있는 결과 제공**
- 통계적 유의성과 함께 **비즈니스적 해석이 필수적**

- 단순 평균 비교만으로는 의사결정을 내릴 수 없다
- 데이터는 항상 변동성을 가지므로 통계적 검정이 필요하다
- p-value뿐 아니라 신뢰구간을 함께 해석해야 한다
